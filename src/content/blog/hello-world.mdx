このブログは [Astro](https://astro.build) と [Bun](https://bun.sh) で作りました。

## なぜ Astro？

Astro は静的サイト生成に特化したフレームワークで、Markdown / MDX でコンテンツを書けます。
Content Collections のおかげで型安全にファイルを管理できるのが気に入っています。

## なぜ Bun？

Node.js / npm の代替として Bun を採用しました。理由は：

- **速い** — インストールもビルドも体感で速い
- **TypeScript をそのまま実行できる** — 余計な設定が不要
- **一つのツールで完結** — ランタイム・パッケージマネージャ・バンドラーが全部入り

## コードブロックのテスト

```typescript
// Bun でも Node.js の API がそのまま使える
const res = await fetch("https://example.com");
const text = await res.text();
console.log(text);
```

```bash
# インストール
bun install

# 開発サーバー起動
bun run dev
```

これからちょこちょこ書いていきます 👋
